Height Calculator from Parents

How This Height Calculator Works

This tool estimates your adult height based on your parents' heights and gender, using the mid-parental height formula.

Formulas:

  • For males: ((father + mother + 5 inches) / 2)
  • For females: ((father + mother - 5 inches) / 2)

Glossary:

Mid-Parental Height
The average of the parents' heights, adjusted by 5 inches for gender.

Disclaimer

This is only an estimate and should not be used as a medical guideline.
