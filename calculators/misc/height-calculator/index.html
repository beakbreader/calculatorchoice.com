Height Calculator from Parents

Understanding Height Prediction

This tool estimates your adult height using the mid-parental height method. It's a simple formula based on your parents' heights and your gender.

Formula:

  • For Males: ((Father + Mother + 5 inches) / 2)
  • For Females: ((Father + Mother - 5 inches) / 2)

Note: If you choose centimeters, the result is automatically converted for you.

Disclaimer:

This is an estimate. Actual height is influenced by genetics, nutrition, and environment.
